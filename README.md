# AXI PRO MAX - منصة الفيديوهات القصيرة الذكية

هذا المشروع هو منصة متطورة للفيديوهات القصيرة مدعومة بالذكاء الاصطناعي (Veo 2.0) وتخزين سحابي ضخم يصل إلى 100MB لكل فيديو.

## 🚀 كيفية نقل المشروع إلى GitHub (نفذ هذه الخطوات الآن)

بما أنك قمت بربط الحسابات، اتبع هذه الأوامر في الـ **Terminal** لنقل الكود:

### 1. إنشاء مستودع جديد
- اذهب إلى [GitHub.com](https://github.com/new) وأنشئ مستودعاً جديداً باسم `axi-pro-max`.

### 2. تنفيذ أوامر النقل (انسخها والصقها في الـ Terminal بالترتيب)
```bash
git init
git add .
git commit -m "إطلاق النسخة الاحترافية AXI PRO MAX"
git branch -M main
git remote add origin https://github.com/USERNAME/axi-pro-max.git
git push -u origin main
```
*(ملاحظة: استبدل USERNAME باسم مستخدم GitHub الخاص بك)*

### 3. التفعيل النهائي على Firebase
- بعد تنفيذ الأوامر أعلاه، اذهب إلى [Firebase Console](https://console.firebase.google.com/).
- اختر مشروعك ثم **Build** -> **App Hosting**.
- ستجد المستودع قد ظهر هناك، اضغط على **Connect** ليبدأ السيرفر بالعمل تلقائياً.

## ✨ الميزات التقنية الحالية
- **الذكاء الاصطناعي (Veo 2.0)**: توليد فيديوهات سينمائية من الأفكار النصية.
- **دعم 100MB**: رفع فيديوهات عالية الجودة مباشرة إلى Firebase Storage مع شريط تقدم سريع.
- **أمان عالي**: نظام تسجيل دخول متكامل مرتبط بقاعدة بيانات Firestore.
